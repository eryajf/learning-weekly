---
- 项目地址：[JsonStudio](https://github.com/sundegan/JsonStudio/blob/main/README_ZH.md)
- 项目说明：一个基于 Tauri 和 Svelte 开发的本地 JSON 工作台，能够进行 JSON 美化、对比、转换和校验，旨在提升开发效率，支持 macOS、Windows 和 Linux 平台使用。

- 项目地址：[panel](https://github.com/acepanel/panel)
- 项目说明：一个基于 Go 语言开发的服务器运维管理面板，能够提供高效运维和离线运行，旨在简化企业级服务器管理，支持主流 Linux 系统使用。
  ![](https://t.eryajf.net/imgs/2026/05/1779610169744.webp)
------
- 项目地址：[momei](https://github.com/CaoMeiYouRen/momei)
- 项目说明：墨梅博客是一个基于 Nuxt 开发的 AI 驱动国际博客平台，能够自动翻译和智能生成内容，旨在为全球创作者提供高效创作体验，支持多种部署方式。
  ![](https://t.eryajf.net/imgs/2026/05/1779610226386.webp)
------
- 项目地址：[metapi](https://github.com/cita-777/metapi)
- 项目说明：一个基于 Node. Js 开发的 AI 中转站聚合网关，能够将分散的多个 API 站点汇聚为一个入口，实现智能路由和成本优化，支持 Docker 和桌面平台部署。
  ![](https://t.eryajf.net/imgs/2026/05/1779610299456.webp)
------
- 项目地址：[arbor](https://github.com/penso/arbor)
- 项目说明：一个基于 Rust 和 GPUI 开发的原生桌面应用，用于运行 agentic 编码工作流，管理 Git 工作树、终端和 diff，支持多平台使用。
  ![](https://t.eryajf.net/imgs/2026/05/1779610373980.webp)
------
- 项目地址：[sub2api](https://github.com/Wei-Shaw/sub2api)
- 项目说明：Sub 2 API 是一个基于 Go 和 Vue 开发的开源 AI API 网关服务，能够统一接入 Claude、OpenAI、Gemini 等模型订阅，支持拼车共享和精确计费，旨在降低 AI 服务使用成本，提供无缝的原生工具集成。
  ![](https://t.eryajf.net/imgs/2026/05/1779610493953.webp)
